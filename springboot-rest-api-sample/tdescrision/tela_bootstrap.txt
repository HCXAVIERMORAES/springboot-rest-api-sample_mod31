Instalar bootsTrap no projeto para fazer as telas
As telas ficam dentro do 'src/maim/resources' -> resources (recursos). Dentro dela criar uma pasta (folder)
de nome static -> por padrão o sprring busca as paginas web nessa pasta de nome static.
Dentro botão direito novo -> html -> index.html
O bootstrap esta na versao 5, procurar a pagin no google e procurar o template inicial.
Criar um formulario utilizando o exmplo dado na 
pagina: https://getbootstrap.com.br/ e fazer as alteraçoes necessarias ao projeto.
atributo do html:
 readonly="readonly" -> desabilita texto no campo
 required="required" -> determina obrigatoriedade do campo
 placeholder="" -> mostra um texto dentro do campo
 
 Ao trabalhar com Rest se faz as se trabalha com ajax, fazendo as requisições por javaScript, puro ou com frameWorks

 Para trabalhar com o JQuery procura-se no google a versão que vai utilizar e no link de CDN utilizar a versão
 minificado, copiar e no projeto acima do bootstrap cola-a. Assim o jquery já esta instalado, para utilizar
 é necessario um evento. Ex: um botão.
 Dentro do botão chamar o evento onclick="salvarUsuario()" -> evento q cham a função javascript salvarUsuario()
 Afunção é criada dentro da tag <script> e fica assim como exemplo:
 		function salvarUsuario() {
			//pegando os elementos da tela
			var id = $("#id").val();
			var nome = $("#nome").val();
			var idade = $("#idade").val();
			
			alert(nome);
		}
		
Ao trabalhar com API se trafega json e a tecnica usada para isso é o AJAX, que é o javascript desparar uma requisição
para o servidor.
É como uma receita de bolo:
	function salvarUsuario() {
			//pegando os elementos da tela
			var id = $("#id").val();
			...			
			//receita de bolo
			$.ajax({
				method: "POST",
				url: "salvar",
				//dados
				data: JSON.stringify({id: id, nome: nome, idade: idade}), //dados da aplicação: dados da tela
				contentType:"application/json; charset=utf-8",
				success: function (response){
					alert("Salvo com sucesso!");
				}
			}).fall(function (xhr, status, errorThrown) {
				alert("Erro ao salvar: " + xhr.responseText);
			});
		}
